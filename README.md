# food-bot
This is a bot that sends a random food emoji when you type `food` in chat.

## Installation

You'll need Python3 and discord(pip) in order to run this file.

## Usage

First, insert your token in line [147 in food.py](https://github.com/Falcn8/food-bot/blob/5b1ebbdffb03115da53dae327767e97a2d60e30d/food.py#L147)

To run this file, just do the following in command prompt or double click the file.
```
cd Downloads
cd food-bot
python food.py
```
